===========
crypto_test
===========

Run:

.. code-block:: console

    $ ./docker_run

Get logs:

.. code-block:: console

    $ ./get_logs

Get report:

.. code-block:: console

    $ ./get_report

Purge logs (including report):

.. code-block:: console

    $ ./purge_logs

Recreate credentials:

.. code-block:: console

    $ ./set_credentials
